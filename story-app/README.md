# The Tortoise and the Hare – React Story Page

A simple React page that shows the story "The Tortoise and the Hare" using three components: Header, Content, and Footer.

## What's in the project

- **Header** – Shows the story title
- **Content** – A short summary in 3 paragraphs
- **Footer** – The moral of the story

## How to run the app

1. Open a terminal and go into the project folder:
   ```
   cd story-app
   ```

2. Install dependencies (first time only):
   ```
   npm install
   ```

3. Start the app:
   ```
   npm start
   ```

4. Open http://localhost:3000 in your browser.
